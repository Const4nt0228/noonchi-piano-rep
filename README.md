# 악보 인식 모듈
사용 기술 
-> APM(Apache, Php, MySQL) 프레임 워크를 사용하여 서버 구축. 
-> PHP를 통해 어플리케이션과 서버의 의사소통 구현
-> 어플리케이션의 요청을 받으면 C++ OpenCV 프로그램이 실행되며 악보 인식
